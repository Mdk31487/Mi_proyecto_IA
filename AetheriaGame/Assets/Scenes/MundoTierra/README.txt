
Módulo 3 - Interacción Social y Reconocimiento de Voz (Juego: Tierra)

Contenido del proyecto:
- Escenario natural abierto con árboles.
- Movimiento libre del personaje.
- Reconocimiento básico de voz para asignación de avatar (niño, joven, adulto).
- Sistema de texto hablado visible para apoyo a la IA.
- Interacciones básicas con NPCs que hablan y reaccionan.
- Entrada con mensaje: "Bienvenido al juego", seguido de la intro narrativa.
- Carpeta lista para continuar desarrollo o cargar en Unity.

Instrucciones:
1. Descomprime este archivo ZIP.
2. Abre la carpeta con Unity Hub (versión recomendada 2021.3+).
3. Espera a que cargue y compila la escena principal.
4. Ejecuta para probar la entrada y recorrido.
5. Personaliza voces o IA desde el script /Scripts/DialogManager.cs

Este módulo está optimizado para PC básicas y tiene preparación para escalar en Google Cloud o PC con GPU.
