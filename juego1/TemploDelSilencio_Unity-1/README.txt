
Templo del Silencio - Módulo Unity

Contenido:
- Assets/Scenes: Escenas de Unity (incluye TemploDelSilencio.unity)
- Assets/Scripts: Scripts base para el mentor, diálogo y entrenamiento
- Assets/Audio: Espacio para sonidos ambientales (viento, agua, etc.)
- Assets/Prefabs: Prefabs de personajes y elementos
- Assets/Art: Arte visual (fondos, UI, decoraciones)
- Assets/Texts: Frases filosóficas, misiones, textos
- Assets/Animations: Movimiento y acciones del mentor
- ProjectSettings: Puedes importar desde un proyecto Unity real

Recomendaciones:
- Usa Unity 2021 o superior.
- Abre el proyecto desde Unity Hub para mantener compatibilidad.
- Comienza importando tu escena base en Assets/Scenes.

Próximos pasos sugeridos:
1. Crear escena del Valle del Espíritu con entorno natural.
2. Importar personajes y asignar scripts de mentor.
3. Agregar sistema de elección de maestro y entrenamiento.
4. Incluir efectos sonoros de ambiente y voz del mentor.
