# Proyecto: Mundo Tierra - Módulo 2

Este proyecto es la continuación del prototipo inicial. Incluye:

- Entorno 3D abierto con árboles y caminos.
- Sistema de roles Niño/Adulto.
- IA con texto en pantalla, preparada para sintetizar voz.
- Detección de palabras clave habladas o escritas.
- Escena de introducción con misión guiada.

## Requisitos

- Unity 2021.3 o superior
- Sistema operativo Ubuntu, Windows o macOS
- 6GB de RAM mínimo (modo ligero)
- GPU integrada o dedicada (modo ampliado para PC potente)

## Instrucciones

1. Abre Unity Hub.
2. Selecciona "Add Project" y carga la carpeta donde descomprimas este zip.
3. Abre la escena ubicada en `Assets/Scenes/MundoTierraModulo2.unity`
4. Ejecuta en modo "Game" y sigue las instrucciones del personaje.

**Nota:** Este módulo está optimizado para hardware limitado, pero tiene estructura modular para escalar.
