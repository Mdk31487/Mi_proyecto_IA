Templo del Silencio Unity Project (with Dialogue & UI Enhancements)

Includes:
- SimpleDialogue script playing speakSound when dialogue starts.
- Option buttons for choices.
- Mentor icon display via UIManager.
- Audio placeholders for speak and selection click.
- Prefabs/UI with placeholder icon.
- Scenes, scripts, audio, prefabs, VFX, IA_Assist structure.

How to use:
1. Open in Unity 2021.3+ via Unity Hub.
2. Ensure AudioManager and UIManager singletons exist.
3. Assign speakSound and optionButtons in SimpleDialogue component.
4. Use SceneLoader to navigate between scenes.
5. Mentor icon will update on dialogue start.
